What this app does

This application helps teachers run engaging vocabulary practice using image uploads, word labels, memory matching, casino-style flashcards, scramble cards, and classroom team scoring.

Main activities

  • Vocabulary Memory: upload pictures, label them, and match each picture to its vocabulary card.
  • Casino Shuffle: draggable, flippable flashcards for suspenseful classroom review.
  • Scramble Cards: spelling practice using synced lesson vocabulary plus Scramble-only extra words.
  • Teams Starboard: create colored teams, award stars, and visualize classroom dollar rewards.

Reward conversion

The current first-place team or tied first-place teams earn classroom dollars at a rate of $1 for every 3 stars, rounded to the nearest dollar. The progress bar shows steady 1/3, 2/3, and full-cycle visual progress toward each 3-star sequence.
